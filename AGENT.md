# AGENT — Punto de entrada

Repositorio educativo sobre **Cursor IDE**: conceptos, reglas, skills, memory, MCP, modelos e integración con skills de Claude.

## Orden de lectura

1. **`README.md`** — Visión general y mapa del repo
2. **`docs/arquitectura-capas.md`** — Tres capas del arnés (contexto, control, orquestación)
3. **`MEMORY.md`** — Cómo funciona la memoria en Cursor
4. **`SKILLS.md`** — Índice de skills del proyecto y rutas de descubrimiento
5. **`docs/graphify.md`** — Mapa visual del repo con graphify
6. **`docs/`** — Guías detalladas por tema (ver tabla en README)

## Convenciones de este repo

- Documentación en **español**
- Ejemplos ejecutables en `.cursor/rules/` y `.cursor/skills/`
- No editar `~/.cursor/skills-cursor/` (reservado por Cursor)

## Tarea típica para un agente

Si el usuario pide aprender o extender este repo:

1. Leer el README y la guía del tema (`docs/0X-*.md`)
2. Proponer cambios mínimos y alineados con ejemplos existentes
3. Actualizar README si añades un concepto nuevo
